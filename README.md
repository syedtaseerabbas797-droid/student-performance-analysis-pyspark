## Project Overview

This project performs **Student Exam Performance Analysis** using **PySpark** and Python.  
The main objective of this project is to analyze student academic performance and identify how different factors affect exam scores.

The analysis includes:
- Gender-based performance comparison
- Impact of parental education
- Effect of lunch type on scores
- Test preparation course analysis
- Subject-wise average score analysis
- Top-performing students identification

This project demonstrates the practical implementation of:
- Big Data Analytics
- PySpark DataFrames
- Data Cleaning
- Data Aggregation
- Data Visualization
